# Hera

Juno emulation synthesizer

## Instrutions

```
git clone --recursive https://github.com/jpcima/Hera.git
cd Hera
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build . -j
```
